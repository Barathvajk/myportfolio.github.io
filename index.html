05 — Security

Security Practices

Security is built into every layer, not bolted on at the end.

🔒
Password Hashing
All passwords hashed using bcrypt with salting. Never stored in plain text.
🪓
JWT Auth
Stateless token-based auth with expiration strategies and protected route middleware.
🛡
Input Sanitization
All user inputs validated and sanitized to prevent injection and XSS attacks.
🌎
CORS Config
Proper Cross-Origin Resource Sharing to control API access by domain.
🗹
Env Variables
DB credentials, JWT secrets, and API keys stored securely — never hardcoded.
🚫
Protected Routes
All sensitive endpoints protected by auth middleware requiring valid JWT tokens.
